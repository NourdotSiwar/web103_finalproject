# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. **Cutter** - A user whose goal is to lose body fat by eating in a calorie deficit while preserving muscle mass through adequate protein intake.
2. **Bulker** - A user whose goal is to gain muscle mass by eating in a calorie surplus with high protein and carbohydrate intake.
3. **Maintainer** - A user whose goal is to sustain their current body weight and composition by hitting a balanced calorie and macro target.

## Draft User Stories

1. As a cutter, I want to set my daily calorie deficit target and macro goals, so that I know exactly how much to eat each day without going over.
2. As a bulker, I want to log multiple large meals throughout the day, so that I can make sure I'm hitting my calorie surplus by the end of the day.
3. As a maintainer, I want to view a dashboard showing my consumed vs. target macros for today, so that I can stay on track without overthinking my diet.
4. As a cutter, I want to add food items to a personal library with their calorie and macro data, so that I can quickly reuse them when building meals.
5. As a bulker, I want to create a meal and add multiple food items to it with specific quantities, so that my meal totals are automatically calculated for me.
6. As a maintainer, I want to edit an existing food item's macro data, so that I can correct mistakes or update values when I buy a different brand.
7. As a cutter, I want to delete a food item from the library that I no longer eat, so that my library stays clean and relevant.
8. As a bulker, I want to see a breakdown of each meal's total protein, carbs, fat, and calories, so that I can quickly tell if a meal is macro-balanced.
9. As a maintainer, I want to browse my meal history by date, so that I can review past eating patterns and make adjustments going forward.
10. As a cutter, I want to see how many calories and macros I have remaining for the day after logging my meals, so that I can plan my next meal accordingly.
11. As a bulker, I want to update my macro and calorie targets when my weight or training changes, so that my goals always reflect my current needs.
12. As a maintainer, I want to reset the app's data back to the default state, so that I can start fresh without having to manually delete everything.
